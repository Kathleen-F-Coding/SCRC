About

Our purpose as the State College Community Recreation Center is to serve local citizens by providing browsable sources that will strengthen the community. We offer many easily accessible resources including support groups, physical therapies, and communal hubs for their leisure. We strive to create a town where relationships between residents, businesses, and students thrive, to create a harmonious, welcoming environment for all to enjoy. While designing our community hub we focused on efficiency for the viewer, so we used softwares including VS Code, and Tailwind CSS to help us with the process. VS Code was our primary programming software used to create the individual code necessary for our website to function, and Tailwind CSS is an extension used within VS Code that helped create individuality to our design.
